		The registration window will display your PC informations (hostid, username,
		hostname, etc ...) on which you will run the STM8 Free Special Edition, which
		are necessary for building the license. You can send these informations by
		either clicking:

		1) "On the Web". The informations are automatically sent with your e-mail address.

		2) "By mail" (if you are using a mail client like Outlook for example) or

		3) "Edit mail". Edit and then send mail (using a mail client like Outlook )

		4) If you don't use mail client, select "Write to file" to write and save
		   the data in a text file, then send it to:

			stm8_free@cosmic.fr

		with "STM32 32K License Request" as subject.
